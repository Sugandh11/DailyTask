Telnet command
Telnet command is used to test the connection to a specific server and port.
or
It is commonly used to test whether a port on a server is open and reachable.
Synatx: telnet <hostname/IP> <port>
Example:
tentet google.co 80
This means you are trying to connect google.com on port 80.
80 → port number (used for HTTP)
If it connects successfully,it means the server is reachable and port is open.

netstat
netstat is a Linux command used to display network connections, open ports, and the state of those connections on a system.
When you run netstat, it shows the status (state) of each network connection.
Important States in netstat
OPEN
A port is available and ready for communication.
LISTEN
A program is waiting for incoming connections on a port.
ESTABLISHED
A connection is active, and two systems are sending and receiving data.
Command: netstat -a 
Above command displays all network connections and listening ports on a Linux system.
